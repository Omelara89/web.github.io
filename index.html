Aprende a programar

Significa adquirir la habilidad de dar instrucciones a una computadora para que realice tareas especificas

Qué implica?

Implica entender la logica de la programación y utilizar lenguajes especificos para crear software ,aplicaciones o resolver problemas de forma automatizada. esencia, es aprender a "hablar" con las maquinas.

Desgloso de conceptos

  1. Lenguaje de programacion
Son sistemas de simbolos y reglas que permite escribir instrucciones que una computadora puede entender y ejecutar.
  • Ejemplo incluyen
    • Python
    • Java
    • JavaScript

    Lógica de programación


    Es el enfoque estructurado y ordenado para resolver problemas utilizando la programación .Implica dividir tareas complejas en pasos más pequeños y secuenciales.
